# Design summary

- Design concept: Outbound caller bullpen
- Layout personality: bold startup
- Typography: Anton headings with Manrope body text
- Color palette: primary #881337, secondary #fb7185, accent #facc15, background #fff1f2, text #1f0710, muted #64748b
- Shape language: Pill-shaped elements
- Visual motif: call waves and contact strips
- Unique layout choices: first sections use Caller bullpen, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
